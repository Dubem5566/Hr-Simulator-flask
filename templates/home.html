Welcome To HR Simulator

A simple and interactive HR system built with Python & Flask for managing employee records and activities.

About The HR Simulator

This HR Simulator is a mini web application that helps managers handle basic HR tasks.
With this system, managers can add employees, view records, approve requests, and manage staff information easily.
This project was built as part of my learning journey in Flask, Jinja2, HTML, and CSS.

Key Features

  • Add New Employees - Register employees with name, department, ID, and salary.
  • View Employee Records - Display all stored employee details in an organized table.
  • Approve or Reject Requests - Handle leave, overtime, or salary-related requests.
  • Delete Employee - Remove specific employee records smoothly.
  • Clock in - Employees can clock in
  • Applications - Employees can apply for vacation, leave or overtime
  • Request Salary - Employees can request for salary
